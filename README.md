# 🛒 쇼핑 리스트 앱

간단하고 깔끔한 쇼핑 리스트 웹 앱입니다. 별도의 설치 없이 브라우저에서 바로 사용할 수 있으며, 데이터는 로컬스토리지에 자동 저장됩니다.

## 주요 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 체크박스로 구매 완료 표시
- 아이템 개별 삭제
- 완료된 항목 일괄 삭제
- 총 개수 / 완료 개수 통계 표시
- 새로고침해도 데이터 유지 (localStorage)

## 사용 방법

1. 입력창에 아이템을 입력합니다.
2. `추가` 버튼 또는 `Enter` 키를 눌러 리스트에 추가합니다.
3. 체크박스를 클릭하면 완료 표시됩니다.
4. `✕` 버튼으로 개별 삭제, `완료 항목 삭제` 버튼으로 일괄 삭제합니다.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage (데이터 영속성)

## 실행 방법

별도 설치 없이 `shopping-list.html` 파일을 브라우저에서 열면 바로 사용 가능합니다.

배포 버전은 Vercel을 통해 제공됩니다.
